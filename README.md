# 🛍️ Smart Retail Demand Forecasting Dashboard

An AI-powered retail analytics platform that predicts future product demand using Machine Learning forecasting models and real retail sales data.

This project combines ARIMA and SARIMA forecasting techniques with an interactive analytics dashboard to help businesses analyze demand trends, identify high-performing products, and make smarter inventory decisions.

---

# 🚀 Project Overview

The Smart Retail Demand Forecasting Dashboard is designed to simulate a real-world enterprise retail intelligence platform.

The system uses:
- Walmart retail sales dataset
- ARIMA & SARIMA forecasting models
- Flask backend
- Tailwind CSS frontend
- Interactive Chart.js visualizations

The dashboard provides:
- Product demand forecasting
- Category-based analytics
- Future sales trend predictions
- Inventory insights
- Dynamic forecast graphs
